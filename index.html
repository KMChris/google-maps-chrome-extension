Re-introduce google maps links to search page

This extension re-adds linked Google Maps button above search results in google search page. In addition, it makes the little map thumbnail on the right side clickable as well

The purpose of this extension is to allow non-technical people (or people afraid to use custom script loader extensions such as Tampermonkey) to have these functionalites back in their chrome browser. The code is extremely simple, all it does it take the current search url and add the missing button/image link with that url address attached
